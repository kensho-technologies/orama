
# Components basic structure

```jsx
<App>
  <Vis>
    <DataList/>
    <DropUI/>
    <Chart/>
  </Vis>
</App>
```

## ChartRender

```jsx
<ChartRender>
  <CanvasRender/>
  <CanvasRenderHighlight/>
  <CanvasRenderHover/>
  <CanvasInput/>
  <RenderAnnotation>
    <Annotation/>
  </RenderAnnotation>
  <AnnotationEditorWrapper/>
</ChartRender>
```

## AnnotationEditor

```jsx
<AnnotationEditorWrapper>
  <AnnotationEditor>
    <Draggable/>
    <ContentEditable/>
  </AnnotationEditor>
</AnnotationEditorWrapper>
```
